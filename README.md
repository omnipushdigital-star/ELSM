# ELSM Portal — Electronic Lock Solution Management

> Comprehensive Managed Service Portal for Electronic Lock Solution as per BSNL RFP.

## 🚀 Quick Start

### Prerequisites
- [Docker Desktop](https://www.docker.com/products/docker-desktop/) running
- [Git for Windows](https://git-scm.com/download/win) installed

### Clone & Run
```bash
git clone https://github.com/YOUR_USERNAME/elsm-portal.git
cd elsm-portal
start.bat
```

That's it. `start.bat` handles everything automatically.

### Access Points
| Service | URL |
|---|---|
| API Docs | http://localhost:8000/api/docs |
| Health | http://localhost:8000/health |
| MinIO | http://localhost:9001 |

### Default Login
```
Email:    admin@elsm.in
Password: Admin@1234
```

## Stop
```bash
stop.bat
```
